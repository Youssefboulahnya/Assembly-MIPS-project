# Assembly-MIPS-project
A simple university project using the assembly language

# About
a simple assembly program that lets visualize the employees of a company by listing them, also lets you search for an employee by his unique ID, calculates and display the max and min of all the salaries, calculates the average salary, calculate and display the number of employees with a salary that exceeds the average salary calculated and last calculates the salary mass of the company.

an employee is represented by their unique ID, a name, a salary.

# How to use
to use this program, install the pcspim.exe program.
